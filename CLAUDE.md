# CLAUDE.md — Autism Detection using ConvNeXt

ASD detection from facial images using the ConvNeXt architecture.

## Approach

ConvNeXt modernization of CNNs — applies vision transformer design principles to convolutional networks for facial classification tasks.

## Key dependencies

- `timm` — PyTorch Image Models library (ConvNeXt implementation)
- `torch`, `torchvision`
- `opencv-python` — image preprocessing
- `scikit-learn` — evaluation metrics

## Running

Load the dataset, run the training notebook/script. Model checkpoints saved as `.pth` or `.pt` files (gitignored).
